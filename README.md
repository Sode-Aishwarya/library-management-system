# Library Management System (Java)

## Project Description

This project is a **simple Library Management System developed using Java**.
It allows users to manage books in a library through a **console-based application**.

The system provides basic functionalities such as:

* Adding books
* Viewing available books
* Issuing books
* Returning books

This project demonstrates the use of **Object-Oriented Programming (OOP) concepts in Java** such as classes, objects, constructors, and ArrayLists.

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Java Collections (ArrayList)
* Scanner class for user input

---

## Project Structure

The project consists of the following Java classes:

### 1. Main.java

Contains the main method and provides the **menu-driven interface** for the user.

Functions included:

* Add Book
* View Books
* Issue Book
* Return Book

---

### 2. Library.java

Handles the **library operations** such as:

* Storing books using ArrayList
* Adding books
* Displaying books
* Issuing books
* Returning books

---

### 3. Book.java

Represents a **Book object** with attributes:

* Book ID
* Title
* Author
* Issue status

Includes a method to display book details.

---

### 4. Member.java

Represents a **Library Member** with attributes:

* Member ID
* Member Name

Includes a method to display member information.

---

## Features

* Add new books to the library
* View all books
* Issue a book to a user
* Return issued books
* Menu-driven console interface

## How to Run the Project

1. Install **Java (JDK 8 or above)**.
2. Clone the repository:

```
git clone https://github.com/yourusername/library-management-system.git
```

3. Navigate to the project folder.

4. Compile the program:

```
javac *.java
```

5. Run the program:

```
java Main
```
